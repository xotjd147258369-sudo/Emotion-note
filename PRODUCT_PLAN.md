# 감정 노트 웹앱 MVP 기획서

## 제품 개요

감정 노트는 개인이 자신의 감정, 감사한 일, 다시 보고 싶은 문장을 차분하게 기록하는 웹앱입니다. 모바일 브라우저에서 설치형 앱처럼 사용할 수 있는 PWA를 기본 경험으로 삼고, 데이터는 Supabase 계정에 안전하게 저장합니다.

## 대상 사용자

- 감정을 짧게라도 꾸준히 기록하고 싶은 개인
- 감사 습관을 만들고 싶은 사용자
- 스스로에게 힘이 되는 문장을 모아두고 싶은 사용자

## MVP 범위

- Google/Kakao 소셜 로그인
- 오늘 기록 요약 홈
- 감정 일기 작성, 수정, 삭제, 목록 조회
- 감사노트 작성, 수정, 삭제, 목록 조회
- 동기부여 문장 카드 작성, 수정, 삭제, 즐겨찾기, 랜덤 보기
- 모바일 우선 UI와 하단 내비게이션
- PWA manifest, 앱 아이콘, 정적 자산 캐시 서비스 워커
- Supabase RLS 기반 사용자별 데이터 격리

## 제외 범위

- 이메일 리마인더
- 브라우저 푸시 알림
- AI 감정 분석과 문장 추천
- 결제, 공유, 커뮤니티, 상담자 대시보드

## 데이터 모델

- `profiles`: 사용자 표시명, 아바타, 시간대
- `emotion_entries`: 감정 점수, 감정 태그, 상황, 본문, 날짜
- `gratitude_notes`: 감사 항목 배열, 메모, 날짜
- `motivation_cards`: 문장, 태그, 테마, 즐겨찾기 여부

## 성공 기준

- 사용자가 소셜 로그인 후 감정 일기, 감사노트, 문장 카드 중 하나를 3분 안에 저장할 수 있다.
- 모바일 화면에서 주요 작성 폼과 하단 내비게이션이 겹치지 않는다.
- 로그인하지 않은 사용자는 기록 화면에 접근할 수 없다.
- 다른 사용자의 데이터는 RLS와 서버 조건으로 접근할 수 없다.

## 확장안

- 주간 감정 리포트
- 기록 내보내기
- 선택적 푸시 알림
- AI 요약과 회고 질문
- 네이티브 앱 래퍼
